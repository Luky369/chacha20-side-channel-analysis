_ChaCha_Capture.ipynb                      : Captures traces for ChaCha20 encryption using fixed or random plaintexts and nonces.
_ChaCha_CPA_Fixed_Nonce.ipynb              : Performs Correlation Power Analysis (CPA) on ChaCha20 encryption with a fixed nonce.
_ChaCha_CPA_Random_Nonce.ipynb             : Conducts CPA on ChaCha20 encryption with random nonces.
_ChaCha_CPA_Random_Nonce_NO_CARRY.ipynb    : Executes CPA on ChaCha20 encryption with random nonces, excluding carry propagation.
_ChaCha_CPA_Random_Nonce_S1.ipynb          : Executes CPA on ChaCha20 encryption on state S1 of ChaCha20 with random nonces.
_ChaCha_CPA_Random_Nonce_S1_NO_CARRY.ipynb : Executes CPA on ChaCha20 encryption on state S1 of ChaCha20 with random nonces, excluding carry propagation.
_ChaCha_DPA_Random_Nonce_XMEGA.ipynb       : Performs Differential Power Analysis (DPA) on ChaCha20 encryption with random nonces on XMEGA.
_ChaCha_NonSpecific-T-Test.ipynb           : Conducts non-specific t-tests on ChaCha20 encryption traces.
_ChaCha_Plot.ipynb                         : Visualizes traces and results from ChaCha20 encryption experiments.
_ChaCha_Specific-T-Test-SICAK-Usage.ipynb  : Executes specific t-tests for ChaCha20 encryption using SICAK.
_ChaCha_Test.ipynb                         : Tests ChaCha20 encryption and decryption for correctness.
_ChaCha20_source.py                        : Contains the implementation of the ChaCha20 encryption algorithm.
SICAK_SPECIFIC_T_TEST.py                   : Implements specific t-tests for leakage analysis using SICAK.
